# Geoscope v2 — デザインリニューアル変更履歴

## 概要
元の `geoscope_complete` をベースに、全ページのデザインシステムを統一・ブラッシュアップしました。

---

## 主な変更点

### デザインシステム (gs-design.css)

#### カラーパレット刷新
| 変更前 | 変更後 | 理由 |
|--------|--------|------|
| `#6366f1` インディゴ蛍光 | `#89c4d4` くすみ水色 | ギラギラ感を排除 |
| `#818cf8` ライトパープル | `#89c4d4` シアン | 色数を統一 |
| `#4ade80` ネオングリーン | `#7bbf96` ミントグリーン | 落ち着いたトーンに |
| `#fbbf24` 黄色 | `#c9a96e` くすみアンバー | Stripe感に寄せる |
| `#f87171` 赤 | `#c47a7a` ダスティレッド | 同上 |
| `#0c0e14` 背景 | `#0d1017` 背景 | ネイビー寄りに |
| `#e8e6e1` テキスト | `#cdd5e0` テキスト | 水色系に統一 |

#### フォント変更
| 変更前 | 変更後 |
|--------|--------|
| Instrument Serif | DM Serif Display |
| JetBrains Mono | DM Mono |
| Inter | Inter（継続） |

- DM Serif Display: よりシャープで現代的なセリフ体
- DM Mono: JetBrains Monoより細身でスマート
- イタリック活用でヒーロー見出しに読ませるリズムを追加

#### UIコンポーネント改善
- ボタン: border-radius 6px に統一（角丸あり）
- インプット: フォーカス色を水色に統一
- バッジ: 蛍光感を排除したくすみトーンに
- スクロールバー: 4px に細く
- トグル: 水色に変更

### 全ページ共通
- nav の高さを 52px → 54px に
- ロゴ: 「Geoscope」シンプルな font-weight:600 に
- ロゴサブテキスト: 「GEO Tracking」に整理
- accent カラー（フォーカス・アクティブ）を水色に統一

### 個別ページ

#### login.html / signup.html / reset-password.html
- カード枠を廃止 → フローティングフォームレイアウトに
- 見出しを DM Serif Display に変更
- フォーカス色を水色に
- エラー/成功カラーをパレットに統一

#### 404.html
- 数字フォントを DM Serif Display に
- バックボタンをプライマリボタンスタイルに

#### index.html / dashboard.html / pricing.html / その他
- フォント・カラー変数を全体適用
- グロー・シャドウ効果を削除
- グリッドの区切り色を薄く調整
